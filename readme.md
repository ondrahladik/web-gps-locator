# Web GPS locator

It is a web application written in html/css/js. 

### The main functions are:
- Latitude, Longitude
- Altitude _(*1)_
- Gridsquare locator
- Map with location 

_*1:_ Altitude is displayed only on mobile. Cannot display altitude on PC.
